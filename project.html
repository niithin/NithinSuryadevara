FindYourPlace.!

This project is a website for Campgrounds across the country where users can register with an account,view campgrounds added by others,add campgrounds individually,delete campgrounds and can perform specific user operations.In order to build the site javascript is used for front-end and node.js,express.js were used for back-end.Further,mongoDB and mongoose were used as database.Passport.js was used to establish the user authentication and authorization functions.The Website was developed across the Groom IDE platform and deployed through HEROKU which is an open source deployment platform.

Shopping Cart.!

This project is “Shopping Cart” using MEAN Stack Technology. Node and Express JS are used in the back end, javascript is used as frontend and MongoDB is used to handle the database. A user has sign up/ login to his/her session in the shopping cart. Once logged in, the user can maintain the cart. However, a user can still view the products list and product detail without logging in. On clicking on a particular product user can view complete details of it. In the product detail page one can rate and review the product. The reviews on other users can be seen under the product description in the product detail page. User can add, delete and edit the products in the cart. In the cart page, user can view the products in the cart, total amount and can select the mode of payment.
